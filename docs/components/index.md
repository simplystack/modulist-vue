## Props
Name       | Type     | Description | Default
---------- | -------- | ----------- | -----
type       | String   | There are 4 types: ['info', 'success', 'warning', 'error'] | 'info'
title      | String   | Title of alert | 'Default title'

## Slots
Name     | Slot props       | Description
-------- | -----------      | -----
default  | None             | Alert content
