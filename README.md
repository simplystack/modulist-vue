# Modulist Vue

Vue components for the Modulist Design System


## Install
```sh
$ npm install --save @modulist/css @modulist/vue
```

## Usage
Include packages in your main.js:
```js
import "@modulist/css/dist/modulist.css";
import ModulistVue from "@modulist/vue";

Vue.use(ModulistVue);
```
