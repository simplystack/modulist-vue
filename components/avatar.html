# Avatar development

Avatars are images that users can set as their profile picture.

# Example

<v-avatar
  src="{{ imageUrl }}"
  alt="My avatar"
  :size="32"
/>

# Props

Name Type Description Default

alt String Image's alt text 'Avatar'
size String, Number Image's size 32
